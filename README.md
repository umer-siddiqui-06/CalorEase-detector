# CalorEase-detector
This mobile app helps users determine the calories, protein, and other nutritional values of their food by uploading an image. It also includes a section with diet plans for both weight gain and weight loss.
## ✨ Features  
✔️ Analyze food images to detect calories, protein, carbs, and other nutrients.  
✔️ Provides personalized diet plans for weight gain and weight loss.  
✔️ User-friendly interface with easy food image uploads.  
✔️ Supports multiple food items in a single image (if applicable).
